# Pixilog ✨

Pixilog is a modern social diary web app where users can capture memories, share public posts, keep private entries, and discover people through a clean pixel-inspired interface.

## 🚀 Features

- User authentication (Register / Login)
- Create, edit, and delete posts
- Public and private posting modes
- Personal diary entries
- Explore public posts and users
- Follow / unfollow users
- Personalized home feed
- Save favorite posts
- Dark / Light mode
- Image uploads with Cloudinary
- Responsive mobile-first design
- Installable PWA experience
- Account deletion support

## 🛠 Tech Stack

- Next.js
- React
- Tailwind CSS
- MongoDB
- Mongoose
- JWT Authentication
- Cloudinary
- Vercel

## 🎨 UI Highlights

- Pixel-inspired branding
- Clean responsive layouts
- Mobile bottom navigation
- Smooth modern interactions
- Light & dark themes
